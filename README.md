# postme
## Meteor app where users can write and read posts
---
### Instructions:
1. Clone this repo  
`git clone https://github.com/vuxor/postme.git`
2. Cd into src directory  
`cd postme/src/`
3. Install dependencies  
`meteor npm install`
4. Start meteor app  
`meteor`

App will create 5 user accounts that you can use to log in.
* username: user[1-5]
* password: postme
